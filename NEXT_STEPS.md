## 📋 ЧТО ДЕЛАТЬ ДАЛЬШЕ:

### 🔹 ШАГ 1: Найти реальное видео на YouTube (2 минуты)

**Через MCP skill:**
```
skill: youtube-search
```

**Промпт:**
```
Найди популярные видео про comedy fails 2024
Нужны видео с большим количеством смешных моментов
Длительность 8-15 минут
```

**Или вручную:**
1. Открой YouTube
2. Поиск: "comedy fails funny moments 2024"
3. Выбери видео с 1M+ просмотров
4. Скопируй URL

---

### 🔹 ШАГ 2: Создать клипы из видео (5 минут)

**Через MCP skill:**
```
skill: youtube-clipper
```

**Промпт:**
```
Создай 3 клипа по 60 секунд из этого видео:
[URL видео из Шага 1]

Таймкоды:
1. 0:15 - 1:15 (смешное падение)
2. 3:30 - 4:30 (неудачный прыжок)
3. 7:00 - 8:00 (реакция кота)

Формат: 9:16 вертикальное для TikTok
```

**Результат:** 3 готовых видео файла в `/root/tiktokproject/output/videos/`

---

### 🔹 ШАГ 3: Опубликовать первый клип (3 минуты)

**Через MCP skill:**
```
skill: tiktok-automation
```

**Промпт:**
```
Опубликуй видео: /root/tiktokproject/output/videos/clip_1.mp4

Описание:
Когда думал что ты крутой... 😅
Парень пытался сделать трюк, но что-то пошло не так
Отметь друга который тоже так падал! 👇

Хэштеги: #fail #comedy #funny #fyp #bike

Время публикации: Сейчас (или запланируй на 18:00)
```

**Повторить для клипов 2 и 3** с соответствующими описаниями.

### 🔹 ШАГ 5: Настроить Telegram уведомления (10 минут)

**1. Создать бота:**
- Открой @BotFather в Telegram
- Команда: `/newbot`
- Имя: "TikTok Notifier"
- Username: "tiktok_notifier_bot" (должен заканчиваться на bot)
- Сохрани токен

**2. Получить Chat ID:**
- Открой @userinfobot в Telegram
- Нажми Start
- Сохрани Chat ID

**3. Настроить в проекте:**
```bash
cd /root/tiktokproject
python scripts/telegram_bot.py setup TOKEN CHAT_ID
```

---

### 🔹 ШАГ 6: Архивировать в Google Drive (1 минута)

```bash
cd /root/tiktokproject
./scripts/sync_drive.sh sync
```

Или через npm:
```bash
npm run sync-drive
```

## 🆘 Если что-то пошло не так:

### Проблема: TikTok API не работает
**Решение:** Проверь авторизацию в Composio
```
skill: tiktok-automation
Промпт: "Help me authenticate with TikTok"
```

### Проблема: Google Trends ошибка 429
**Решение:** Слишком много запросов, подожди 1 час

### Проблема: rclone не синхронизирует
**Решение:** Перенастрой Google Drive remote
```bash
rclone config
→ gdrive → Reconfigure
```
