## 🚀 КАК ИСПОЛЬЗОВАТЬ:

### Вариант 1: Загрузить напрямую в TikTok

1. Скопируй видео файлы
2. Открой TikTok
3. Загрузи каждое видео
4. Добавь музыку из библиотеки TikTok
5. Опубликуй

### Вариант 2: Добавить музыку через ffmpeg

```bash
cd /root/tiktokproject/output/videos/final
ffmpeg -i video_01_ego.mp4 -i music.mp3 -c:v copy -c:a aac -shortest output.mp4
```
